EventManager REST Web Service
-----------------------------

1-Deploy eventmgr.war into Tomcat 7.

2-place fitman folder into the user home directory of the system.

	-Linux:   /home/username
	 
	-Windows: c:\users\username

	-This folder contains the event.properties file. Edit this file and change the IP value using the 
	IP of the Virtual Machine of SEM Test or the IP where is located SEM.
